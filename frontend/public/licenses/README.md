# Licences Gaulois

Déposer ici les licences / attributions pour chaque asset embarqué dans la page Gaulois.

## Exemple de structure
- `theme-music.txt` – source, licence, URL de la piste audio (`theme.mp3`).
- `logo-fr.txt` – attribution/mention si un logo externe est importé.

Format recommandé :
```
Nom asset : theme.mp3
Auteur    : Votre Nom ou Auteur original
Licence   : CC0 1.0 / CC-BY 4.0 …
Source    : https://…
Date      : 2024-..-..
Notes     : (facultatif)
```

> Maintenir ces fichiers à jour est indispensable pour la conformité juridique avant déploiement.
